Ubuntu-22.04.3 server 树莓派4

1.准备工作

Ubuntu Server系统（默认已经切换为root用户）——使用安装Ubuntu22.04.3 LTS Server版的树莓派4
EC20/EC25/EG25模块/开发板（USB接口）——USB接口EC20模块
4G手机卡——中国电信手机卡
